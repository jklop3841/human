# Worldview

这里保存经过结构化的正式观点。每个观点必须拥有唯一 ID、版本、状态、适用边界、反方观点和 Agent 使用说明。

## 建议主题目录

- `meta/`：Human 项目、认知保存与数字人格
- `human/`：人性、希望、欲望与命运
- `ai/`：AI、人机关系与治理
- `china/`：中国社会观察
- `economy/`：财富、劳动与交易
- `state/`：国家、权力、战争与秩序
- `gender/`：性别、婚恋与家庭
- `narrative/`：叙事、身份与信任
- `ethics/`：伦理、责任与边界

新增内容优先使用 `templates/viewpoint.md`。
