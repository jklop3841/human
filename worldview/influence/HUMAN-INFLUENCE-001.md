# 核心主张

真正有效的说服，不是把一个观点塞进人脑，而是进入一个人已经存在的认知轨迹，通过场景、线索、情绪和矛盾改变轨迹方向，让他自己发现、说出并认领新的结论。

## AI默认写作的问题

AI经常采用“观点—证据—案例—总结—号召”的证明题结构。即使语气温柔，只要作者预先垄断结论，读者仍然只是被教育的对象，这就是AI文章常见的说教味。

认知轨迹式写作把文章改造成一次发现：

```text
进入读者已有生活
→ 识别尚未语言化的矛盾
→ 放入旧解释无法处理的异常
→ 让真实线索逐渐改变方向
→ 保留最后一块推理空间
→ 读者自行形成新解释
```

## 核心概念

### 认知轨迹

人在阅读前已经形成的暂时解释系统，包括生活经验、身份、欲望、恐惧、羞耻边界、未来想象与行动成本。

### 认知裂缝

轨迹中尚未解决的冲突，例如想改变却害怕行动、拥有经验却无法兑现价值、相信机会又害怕被骗。

### 轨迹修正

通过真实场景、反常细节和侧面证据，使旧解释逐渐失去稳定性，让读者主动寻找更能解释自身经历的新框架。

### 自我归因式说服

作者不直接灌输结论，而是设计足够真实的认知现场，让读者自行补全关键结论并保留解释权。

### 侧面证明

不直接声明品质，而是展示品质在现实中长期留下的结果，让读者从结果反推原因。

## 三个作用层

1. **身心状态层**：文字经过意义理解、记忆和处境匹配，影响情绪、注意力与行动准备。身体是物质底座，但不是可被文案直接寻址的激素或脑区接口。
2. **观点解释层**：改变人如何解释自己和现实，例如从“我没有价值”转向“我的经验与旧岗位的定价方式发生错位”。
3. **行为生成层**：在观点变化后增加可欲目标、可见路径、主体能力感、可承担成本、明确触发器和最小行动。

## 希望与恐惧

可行动的希望由“可欲目标、可见路径、主体能动感”共同构成。只有美好画面而没有路径，是虚假希望。

恐惧主要驱动中止、逃离和防御。负责任的威胁表达必须提供真实证据、有效出口和选择权；只制造焦虑不提供出口属于操控。

## 证据使用原则

证据不是用于砌出不可反驳结论的砖块，而是改变读者解释方向的线索。每条证据至少应完成一种功能：

- 暴露旧解释中的异常；
- 排除错误归因；
- 呈现事实之间的冲突；
- 重新解释过去经历；
- 暗示新方向；
- 为读者留下可补全的最后一步。

## 完整影响路径

```text
可识别生活场景
→ 未说出口的矛盾
→ 错误羞辱被移除
→ 反常线索进入
→ 旧解释产生裂缝
→ 读者自行重构归因
→ 新观点与身份连接
→ 出现可信希望或避险出口
→ 看见现实路径
→ 执行最小行动
```

## 反方观点与限制

- 并非所有受众都拥有相同轨迹，同一内容可能产生相反反应。
- 观点变化不必然导致行为变化。
- 希望和恐惧不是仅有的行动动力。
- 轨迹设计仍可能被用于操控，隐藏结论不等于自动获得正当性。
- 对受众内心状态的判断可能只是作者投射，必须通过真实反馈校准。
- 侧面证明仍可能被选择性剪裁，因此必须保持事实完整。

## Agent使用说明

允许：

- 将观点、商业信息、教育内容和社会观察改写为非说教式叙事；
- 建模读者已有轨迹、裂缝、尊严边界和行动阻力；
- 使用场景、线索、推理缺口、希望路径与侧面证明；
- 分别设计观点改变和行为改变。

禁止：

- 伪造事实、经历、权威、共识、稀缺和案例；
- 利用疾病、贫困、孤独、羞耻或依赖进行隐蔽操控；
- 制造敌人强迫站队；
- 用无出口恐惧推动服从；
- 宣称文字可以精准控制激素、脑区、男性或女性；
- 为了转化率剥夺读者的知情、比较、选择和退出权。
