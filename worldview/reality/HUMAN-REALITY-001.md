# HUMAN-REALITY-001：脏现实结构场论

## 一句话定义

现实结果不是由单一观点或单一动作线性产生，而是行动者的意图进入一个由关系、资源、权限、信息差、叙事、反馈、维护成本和冲击组成的动态结构场后，共同碰撞形成。

这是卢成的个人现实建模框架。它使用“场、质量、热力学、熵、弹球、吸引子”等比喻帮助推理，不主张这些比喻本身构成物理学或社会科学证明。

## 核心命题

### 命题一：关系先于孤立对象

对现实行动而言，一个主体“是什么”往往不足以解释结果，还要知道它与其他主体之间存在什么交换、依赖、控制、信任、竞争、许可和信息差。

核心链条：

`关系 → 结构 → 场 → 路径 → 结果`

### 命题二：结构改变可行路径

关系形成相对稳定的排列后，会产生机会、阻力、诱因和边界。纸面上存在的选择，不等于行动者现实中拥有的选择。

### 命题三：高影响因素会弯曲路径

钱、现金流、权力、信任、叙事、平台、工具、身体、时间、法律身份、组织资源、社会关系和执行权限，会显著改变行动的成本、收益、风险和成功概率。框架把它们比喻为“现实质量体”。

### 命题四：偏差可能暴露隐藏结构

预期路径和实际路径之间的稳定偏差，可能提示尚未观察到的信息或关系。但隐藏变量必须通过可观察预测验证，不能由偏差直接推导出阴谋、恶意或秘密协调。

### 命题五：现实行动是多线碰撞

计划包含一条主意图线，现实还包含干扰线、执行摩擦、反馈延迟和突发冲击。结果是多条线的交互，而非意图的直接投影。

### 命题六：系统需要持续维护

组织、流程、关系和知识在缺少投入、纠错、反馈、边界维护和更新时会衰退、漂移或腐化。所谓“脏现实热力学”是对这种维护需求的启发式描述。

### 命题七：人会进入事件世界

人的行动会受到叙事、权威、群体反馈、时间压力、早期奖励、沉没成本和未知信息的共同塑造。这些因素可能形成吸引子和路径依赖，使后续行为越来越难脱离既有结构。

## 基本对象

| 对象 | 定义 | 必须落地为 |
|---|---|---|
| 主体 | 能行动、响应或否决的个人、群体、机构、平台 | 目标、资源、权限、约束 |
| 关系 | 主体间的交换、控制、依赖、信任、竞争、许可与信息差 | 方向、强度、时效、证据 |
| 结构 | 一段时间内相对稳定的关系排列 | 哪些路径被打开或关闭 |
| 场 | 结构对行动产生的机会、阻力、诱因与边界 | 成本、概率、权限或风险变化 |
| 现实质量体 | 显著弯曲可行路径的因素 | 具体影响变量 |
| 主意图线 | 从行动到预期结果的核心因果路径 | 每一步的必要条件 |
| 结果干扰线 | 有独立证据、会改变行动结果的因素 | 因果链与可证伪预测 |
| 诊断干扰 | 缺失、陈旧或失真的测量使原因无法归属 | 数据缺口及其修复办法 |
| 暗信息 | 未观察、但可能改变关系结构的变量 | 候选假设与验证办法 |
| 反例 | 在模型适用条件内违背核心预测的结果 | 需要修订或废弃的命题 |
| 事件世界 | 多种社会与时间因素共同形成的行动环境 | 进入、强化、退出机制 |

## 三个子模型

### 暗信息质量模型

推理顺序：

`路径偏差 → 候选暗信息/暗关系 → 竞争解释 → 可观察预测 → 验证或驳回`

暗信息只能解释它能够产生独特预测的部分。若任何结果都能被它解释，它就没有分辨能力。

### 脏现实弹球模型

行动路径会受到政策、平台、现金流、工具、合作关系、身体状态与偶然事件的碰撞。模型要求使用条件化预测、情景区间、转向阈值、冗余和熔断，而非假定直线执行。

### 脏现实热力学

用以下可观察量替代空泛的“熵”：

- 维护频率；
- 反馈延迟；
- 错误与任务积压；
- 数据陈旧度；
- 权责错位；
- 人员流失；
- 现金跑道；
- 规则例外数量；
- 恢复所需时间。

## 反例与干扰的裁决规则

1. 行动前尽可能登记已知干扰及其判断信号。
2. 结果违背核心预测时，先将其视为潜在反例。
3. 只有存在独立证据和可信因果路径时，才升级为结果干扰。
4. 测量缺失只能暂列为诊断干扰，不能自动解释业务结果。
5. 事后提出的干扰必须产生新的可验证预测。
6. 无法裁决时标记为未知，不强行保护或推翻模型。
7. 同类偏差反复出现而无法被模型预测时，应收缩适用范围或修订核心命题。

## 适用范围

适合用于：

- 组织、商业、传播、个人项目和公共事件的结构诊断；
- 分析计划为何偏离执行；
- 寻找权限、资源、信息和反馈的瓶颈；
- 设计小测试、冗余和止损；
- 为 Agent 提供现实推理的对象表。

不适合用于：

- 把比喻包装成科学定律；
- 在没有证据时推断阴谋或他人内心；
- 事后为任何失败辩护；
- 代替法律、医学、金融或安全领域的专业证据；
- 为操控、胁迫或伤害他人提供合理化。

## 主要质疑

### 质疑一：概念过宽，什么都能装进去

回应：每个概念都必须落到具体变量、证据、因果路径和拒绝条件。无法操作化的部分只保留为隐喻。

### 质疑二：暗信息会使理论不可证伪

回应：暗信息只能作为候选假设，必须与竞争解释比较，并写出使其被拒绝的观察。

### 质疑三：把失败叫作干扰是在逃避错误

回应：默认优先考虑潜在反例。干扰必须预先登记，或拥有与结果独立的证据和可验证的因果链。

### 质疑四：借用物理概念会造成伪科学

回应：“场、质量、热力学、弹球”只承担认知压缩功能。正式分析必须翻译成关系、成本、权限、概率、反馈和维护指标。

### 质疑五：结构解释会忽略个体能动性

回应：结构限制路径，但不决定全部结果。模型同时记录行动者的目标、选择、学习、结盟和改变结构的能力。

## 修订条件

出现以下情况应修订本理论：

- 关键概念无法稳定转化为可观察变量；
- 使用框架没有比简单基线产生更好的预测或决策；
- 干扰分类长期依赖事后解释；
- 同类案例中结构变量与结果不存在稳定关系；
- 新证据表明个体特质、随机性或其他机制解释力更强。
